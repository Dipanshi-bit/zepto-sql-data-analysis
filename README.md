
This SQL script covers:

- **Table creation and schema setup** for Zepto product data
- **Data exploration**: row counts, sample data, null checks, and distinct categories
- **Data cleaning**: identifying zero prices and converting prices from paise to rupees
- **Business questions analyzed**:
  1. Top 10 products by discount percentage
  2. High MRP products that are out of stock
  3. Estimated revenue by product category
  4. Products with MRP > 500 and discount < 10%
  5. Top 5 categories by average discount percentage
  6. Price per gram for products > 100g, sorted by best value
  7. Categorizing products by weight (low, medium, bulk)
  8. Total inventory weight by category

This project demonstrates business-driven data analysis skills using SQL for retail pricing and inventory insights.
